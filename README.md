
# serve

> A simple zero dependency reload server.

## Install

This project is only hosted on Github. Fortunately, you can still install via npm.

```
npm i whaaaley/serve
```

## Use

```
serve --scss "npm run scss" --js "npm run js" --watch "src"
```

## Command Line Flags

```
--<extension> "<command>"
--watch "<directory>"
```
